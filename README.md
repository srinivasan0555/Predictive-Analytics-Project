# Predictive Analytics Project

## Description
This project predicts store sales using Machine Learning (Linear Regression) in Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Store Sales - Time Series Forecasting (Kaggle).

Due to GitHub web upload size limits, the dataset is not included in this repository.

## How to Run
1. Install the required Python libraries.
2. Place the dataset in the project folder.
3. Run:
   python predictive_analytics.py
